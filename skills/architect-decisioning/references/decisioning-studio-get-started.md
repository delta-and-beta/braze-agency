# Decisioning Studio Getting Started

BrazeAI Decisioning Studio™ deploys decisioning agents that optimize business metrics through personalized customer engagement.

## Key Design Decisions

Work with the AI Decisioning Services team to define:

| Decision | Description | Examples |
|----------|-------------|----------|
| **Success metric** | What the agent maximizes | Revenue, LTV, ARPU, conversions, retention |
| **Audience** | Who the agent makes decisions for | All customers, loyalty members, at-risk subscribers |
| **Experiment groups** | Structure of randomized controlled trials | Decisioning Studio, Random Control, BAU, Holdout |
| **Dimensions** | What aspects to personalize | Time of day, subject line, frequency, offers, channel |
| **Options** | Available choices for the agent | Specific templates, offers, time windows |
| **Constraints** | Decisions the agent must never make | Geographic restrictions, budget limits, eligibility rules |

These decisions directly affect the incremental uplift achievable and the speed at which value is generated.

## Capabilities

| Capability | Details |
|------------|---------|
| **Any success metric** | Revenue, conversions, ARPU, LTV, or any business KPI |
| **Unlimited dimensions** | Offer, channel, timing, frequency, creative, and more |
| **Any CEP** | Native integrations with Braze, Salesforce Marketing Cloud, or custom |
| **AI Decisioning Services** | Dedicated Braze data science team support |
| **Advanced experiment design** | Fully customizable treatment groups and holdouts |

## Best Practices

- **Maximize data richness** — More customer data leads to better agent performance.
- **Diversify actions** — A broader action set enables more granular per-user personalization.
- **Minimize constraints** — Constraints should enforce business rules only; avoid over-constraining to allow effective experimentation.
