## Canvas Export Overview

Canvas user data can be exported as CSV from either the full Canvas or individual components.

### Export full Canvas data

1. Go to **Messaging > Canvas** and select your Canvas.
2. In the **Canvas Details** section, open the **User Data** dropdown.
3. Choose an export option:
   - **CSV Export User Data** — all user data for Canvas entrants
   - **CSV Export Email Address** — email addresses only

### Export component data (original workflow only)

Per-component export is available in the original Canvas workflow only:

1. Select the specific component.
2. Open the **User Data** dropdown on the **Canvas Step Details** page.

### Troubleshooting

For CSV and API export issues, see the [export troubleshooting](../export_troubleshooting/) article.
