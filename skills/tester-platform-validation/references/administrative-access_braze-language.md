## Dashboard Language

Change the Braze dashboard display language via the globe icon in the global header.

**Steps:**
1. Select the globe icon (`fa-globe`) in the global header
2. Choose your language from the list

The dashboard will reload after a few moments.

**Supported languages:**

| Language | Native Name |
|----------|-------------|
| German | Deutsch |
| English | English |
| Spanish | Español |
| French | Français |
| Japanese | 日本語 |
| Korean | 한국어 |
| Portuguese | Brasil |

> Note: Dashboard language and notification language are configured separately. Notification language settings are pending development.
