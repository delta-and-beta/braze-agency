## Worthy Visual Content

Worthy integrates with Braze to deliver personalized, rich in-app experiences built with a drag-and-drop editor. It auto-generates Connected Content servers, secured APIs, and analytics-enabled HTML for use in Braze Custom Code campaigns.

**Use cases:** Custom onboarding welcomes, promotional experiences, feedback/ratings collection, product idea testing, rich notices and community updates.

**Prerequisites:**
- Worthy account
- Braze SDK configured in your mobile app

---

### Integration Steps

**1. Build message in Worthy**
In the Worthy dashboard, navigate to your app → **Message Creator** → create your personalized message.

**2. Create Braze campaign**
Create an in-app message campaign in Braze. Set **Message Type** to **Custom Code**.

**3. Export and paste**
In Worthy's message creator: **Export** → **Braze** → copy the exported HTML into the **HTML + Asset Zip** text box in the Braze campaign editor.

Test immediately via the **Test** tab in the Braze campaign editor.

---

**Key capabilities auto-configured by Worthy:**
- Connected Content server and secured API
- Analytics and click-tracking surfaced in Braze
- Dynamic content and required API connections embedded in exported HTML
