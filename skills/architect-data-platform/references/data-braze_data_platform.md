# Braze Data Platform Overview

Braze Data Platform is a composable set of data capabilities and partner integrations built around three core jobs-to-be-done:

- **Data Unification** — collect and consolidate first-party data
- **Data Activation** — clean, segment, and act on that data in real-time
- **Data Distribution** — stream and export data to external systems

## Data Unification

Bring data into Braze from any source:

- **APIs and SDKs** — programmatic ingestion from any platform
- **Cloud Data Ingestion (CDI)** — direct integration from data warehouses or file storage
- **Data Transformation** — build and manage webhook integrations that pipe external data into Braze

## Data Activation

Prepare data for targeting and personalization:

- Build real-time **user profiles** capturing behaviors and preferences
- Create **segments** to target specific customer cohorts
- Use the **Reports Metric Glossary** to measure how customers respond to personalized experiences

## Data Distribution

Export Braze data to downstream systems:

- **Braze Currents** — streams raw Braze event data to a data warehouse, enabling BI tools and custom analytics
- Use exported data for next-step insights, attribution, and cross-platform decisions

## Key Takeaway

The three layers form a data loop: unify raw data → activate it for personalization → distribute outcomes for broader analysis. Each layer has dedicated tooling and the layers are independently composable.
