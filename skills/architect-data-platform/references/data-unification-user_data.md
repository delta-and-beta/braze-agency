## User Data Management

### Dummy User Blocking

Braze automatically bans or blocks users ("dummy users") that exceed **5 million sessions** with no SDK event ingestion. This typically indicates a misintegration.

**If a legitimate user is blocked:** Contact your Braze account manager to resolve the issue.
