## Connected Content Public APIs

A reference list of publicly available APIs compatible with Braze Connected Content. Note that public APIs may have usage restrictions and rate limiting — review API documentation and contact providers before use.

### News & Information

| API | Description |
|-----|-------------|
| [OpenWeatherMap](http://openweathermap.org/api) | Current weather, 5/16-day forecasts, and historical weather data |
| [NYT Article Search](https://developer.nytimes.com/docs/articlesearch-product/1/overview) | NYT article data: headline, topic, URL, date, abstract |
| [The Guardian API](http://open-platform.theguardian.com/documentation/) | Guardian article data: headline, topic, URL, date, abstract |

### Events & Ticketing

| API | Description |
|-----|-------------|
| [SeatGeek](http://platform.seatgeek.com/) | Ticket info for concerts, sports, and theater |
| [OnConnect](http://developer.tmsapi.com/docs/read/data_v1_1/movies/movie_showtimes) | Box-office showtimes for US and Canadian theaters |
| [Eventbrite](http://developer.eventbrite.com/) | Public event data |
| [Eventful](http://api.eventful.com/) | Public event data |
| [Ticketmaster](http://developer.ticketmaster.com/products-and-docs/apis/getting-started/) | Events, venues, and pricing |

### Food & Drink

| API | Description |
|-----|-------------|
| [BreweryDB](http://www.brewerydb.com/) | Breweries, beers, and beer events |

### Finance

| API | Description |
|-----|-------------|

### Music

| [Songkick](http://www.songkick.com/developer) | Live music: artists, venues, locations |
