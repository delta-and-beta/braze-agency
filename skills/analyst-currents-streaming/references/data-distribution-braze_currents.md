## Braze Currents Overview

Currents is Braze's event streaming feature for exporting real-time behavioral data to external systems.

### Capabilities

- Stream Braze event data to a **data warehouse** or **analytics partners** for analysis
- Power **BI tools**, **ML algorithms**, and other downstream systems with continuous event streams
- Route event data through CDPs: **Tealium**, **Segment**, or **mParticle**

### Access & Setup

- Included in **pro and enterprise-level packages**
- Contact your **account manager** to enable; they can assist with setup and integration
- Data specialists are available to support the integration process

### Use Cases

- Detailed campaign and user behavior analysis
- Real-time data pipelines for business intelligence
- Feeding ML models with behavioral signals
- Cross-platform event routing via CDP middleware
