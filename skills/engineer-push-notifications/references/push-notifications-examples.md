# Push Notification Examples

Advanced push notification implementation examples for the Braze SDK, covering Android and Swift (iOS) platforms.

## Platform Coverage

| Platform | SDK |
|----------|-----|
| Android | Braze Android SDK |
| Swift (iOS) | Braze Swift SDK |

## Android

Refer to the Braze Android SDK developer guide for advanced push notification examples, including:

- Custom notification layouts
- Action buttons
- Rich notifications (images, expanded views)
- Notification channels
- Silent/background push handling

## Swift (iOS)

Refer to the Braze Swift SDK developer guide for advanced push notification examples, including:

- UNNotificationServiceExtension for rich push
- UNNotificationContentExtension for custom UI
- Push action buttons
- Badge count management
- Background push handling
