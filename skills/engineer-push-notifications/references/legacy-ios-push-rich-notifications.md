# iOS Rich Notifications

iOS 10+ supports push notifications with images, GIFs, and video via a **Notification Service Extension** â€” a binary bundled with your app that intercepts and modifies push payloads before display.

## Setup

### 1. Create the Service Extension
In Xcode: **File > New > Target > Notification Service Extension**. Set **Embed In Application** to embed it in your app.

### 2. Configure Bundle ID & Provisioning
Register a separate App ID and provisioning profile in the Apple Developer Portal. The extension's bundle ID must differ from your main app's:
- Main app: `com.company.appname`
- Extension: `com.company.appname.AppNameServiceExtension`

## Braze Payload Structure

Braze sends rich content under the `ab` key in the APNs payload:

```json
{
  "ab": {
    "att": {
      "url": "http://mysite.com/myimage.jpg",
      "type": "jpg"
    }
  },
  "aps": { ... }
}
```

**Key constants (Objective-C):**

```objc
static NSString *const AppboyAPNSDictionaryKey           = @"ab";
static NSString *const AppboyAPNSDictionaryAttachmentKey = @"att";
static NSString *const AppboyAPNSDictionaryAttachmentURLKey  = @"url";
static NSString *const AppboyAPNSDictionaryAttachmentTypeKey = @"type";
```

**To display manually:** download the file from `url`, save with the `type` extension, and add to notification attachments.

## Example Implementation

Replace the autogenerated `NotificationService` file in your extension target:

- **Objective-C:** use [`NotificationService.m`](https://github.com/Appboy/appboy-ios-sdk/blob/master/Example/StopwatchNotificationService/NotificationService.m)
- **Swift:** use [`NotificationService.swift`](https://github.com/Appboy/appboy-ios-sdk/blob/master/HelloSwift/HelloSwiftNotificationExtension/NotificationService.swift)

## Creating Rich Notifications in Braze Dashboard

1. Create an iOS push campaign
2. Attach an image/GIF directly, or provide a hosted URL (image, GIF, or video)

**Note:** Assets download on push receipt â€” plan for synchronous traffic spikes if self-hosting.

Refer to [`UNNotificationAttachment`](https://developer.apple.com/reference/usernotifications/unnotificationattachment) for supported file types and size limits.
