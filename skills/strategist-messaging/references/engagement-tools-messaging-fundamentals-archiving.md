## Archiving Campaigns and Canvases

Archive campaigns/Canvases to reduce workspace clutter without losing results data.

### Archiving

**Effect:** Stops message delivery, hides in-app messages, blocks editing. Results remain viewable.

**To archive a campaign:**
1. Go to **Messaging** > **Campaigns**
2. Hover over the campaign row → select `⋮` menu
3. Select **Archive**

**To archive a Canvas:**
1. Go to **Messaging** > **Canvas**
2. Hover over the Canvas row → select `⋮` menu
3. Select **Archive**

### Unarchiving

**Effect:** Status changes to **Active** only — does not resume sending. Safe for editing/reviewing without triggering messages.

**To unarchive:** Select `⋮` menu on the row → **Unarchive**

**To resume sending after unarchiving:** Select `⋮` menu → **Resume**

### Key Behaviors

| State | Messages sent | Editable | Results visible |
|-------|--------------|----------|-----------------|
| Archived | No | No | Yes |
| Unarchived (Active) | No | Yes | Yes |
| Resumed | Yes | Yes | Yes |
