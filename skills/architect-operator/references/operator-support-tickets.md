## Operator Support Tickets

File support tickets directly from BrazeAI Operator without leaving the Braze dashboard.

### Filing a Ticket

1. Ask Operator a question, then select **File a support ticket** (or ask Operator to file one directly).
2. Fill in the title and description.
3. Review suggested attachments and context.
4. Select **Submit**.

### Auto-Included Context

Braze automatically attaches:
- Recent Operator chat messages
- Page context from the active page when Operator was opened
- Workspace name
- Relevant logs and session details

Additional context can be added manually via text field or file attachments.

### Tips for Faster Resolution

- Describe exact steps to reproduce the issue.
- Include error messages or a description of unexpected behavior.
- Review auto-suggested attachments before submitting.
