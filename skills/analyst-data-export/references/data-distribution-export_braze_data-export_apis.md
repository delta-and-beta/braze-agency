## Export APIs

Export APIs allow programmatic export of dashboard data as JSON files.

**Key use cases over CSV dashboard export:**

- **File size**: Dashboard CSV exports are capped at 500,000 rows. The Export API has no row limit.
- **Programmatic access**: When you need to interact with data in code rather than a spreadsheet.

**Reference:** See [Export endpoints](https://www.braze.com/docs/api/endpoints/export/) for available data, parameters, and sample code.

**Troubleshooting:** For CSV and API export issues, see [Export troubleshooting](https://www.braze.com/docs/user_guide/data/export_braze_data/export_troubleshooting/).
