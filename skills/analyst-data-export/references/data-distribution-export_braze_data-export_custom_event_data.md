## Export Custom Event Data

The **Custom Events** report (under **Analytics**) displays occurrences of one or more custom events over time, with optional segment filtering.

### Available CSV Exports

| Export | Segment Filter Available |
|--------|--------------------------|
| Custom Events by Date | Yes |
| Custom Events by Hour | Yes |
| Custom Events per MAU | No |
| Custom Events for Different Segments | No |
| Custom Events per KPI Formula | Yes |

### Notes

- Hourly and date-based views support drilling down by specific audience segments.
- For export issues, see the Export Troubleshooting guide.
