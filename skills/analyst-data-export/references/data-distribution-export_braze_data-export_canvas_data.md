## Export Canvas Data

Export user data from a Canvas or individual Canvas component as CSV.

### Export data for an entire Canvas

1. Go to **Messaging** > **Canvas** and select your Canvas.
2. In the **Canvas Details** section, select the **User Data** dropdown.
3. Choose an export option:
   - **CSV Export User Data** — exports all user data for Canvas entrants
   - **CSV Export Email Address** — exports email addresses only

### Export data for a specific component (original workflow only)

1. Select the specific Canvas component.
2. On the **Canvas Step Details** page, select the **User Data** dropdown.

> Note: Per-component export is only available in the original Canvas workflow, not the current workflow.

### Troubleshooting

For CSV and API export issues, see the export troubleshooting guide.
