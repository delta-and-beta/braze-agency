## Exporting App Usage Data

From the **Home** dashboard, you can export CSV files of app usage statistics.

### Steps to Export

1. Set the **time frame** and **apps** (defaults to last 30 days, all apps).
2. Scroll to the **Performance Over Time** graph.
3. In the **Statistics For** field, select the data to export.
4. Click the chart context menu icon and select your export option.

### Available Export Data

| Category | Metrics |
|----------|---------|
| **Sessions** | Session Count by Date (optionally by segment or app version), Session Count by Hour, Session Count per MAU by Date |
| **Users** | DAUs by Date (optionally by segment), MAUs by Date, New Users by Date, Stickiness by Date |
| **Email** | Sent, Delivered, Opened, Clicks, Bounces, Reported as Spam — all by Date |
| **Push Notifications** | Sent, Total Opens, Direct Opens, Bounces by Date (optionally by platform) |
| **In-App Messages** | Sent, Impressions, Opened — all by Date |

### Notes

- All exports are CSV format.
- For export errors, see the [export troubleshooting guide](https://www.braze.com/docs/user_guide/data/export_braze_data/export_troubleshooting/).
