## iOS Initial SDK Setup

The Braze iOS SDK provides basic analytics (session handling) and in-app messages out of the box. Additional channels and features require further integration steps.

### Installation Methods

- **CocoaPods**
- **Carthage**
- **Swift Package Manager**
- **Manual integration**

RubyMotion apps are fully supported.

### Size Impact

Adds **1–2 MB** to the app IPA, plus an APP file and **~30 MB** for the framework.

### Next Steps

1. Complete one of the installation methods above
2. Follow the [completing the integration](#) steps
3. Optionally enable additional SDK customizations
4. Integrate channels and features for your campaigns

> **Note:** The Objective-C SDK is deprecated. Prefer Swift-based integration paths where possible.
