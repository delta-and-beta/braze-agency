## Email Channel Index

### Customization Options

- **Braze email templates** — pre-built templates for quick setup
- **Custom HTML templates** — full HTML control for branded designs
- **Drag-and-drop editor blocks** — visual block-based composition
- **User subscriptions** — manage opt-in/opt-out states

### Use Cases

| Use Case | Description |
|---|---|
| Re-engagement | Reach users outside the app, including non-installers |
| Onboarding | Encourage push opt-ins or social sharing for new users |
| Rich messages | Dynamic HTML content with full formatting support |
| Multimedia | Embed videos and images to drive engagement |
| Newsletters | Weekly/monthly sends to maintain long-term engagement |
| Transactional | Purchase confirmations, shipping updates via transactional email API |

### Notes

- Transactional emails are sent via a dedicated API campaign type (separate from standard marketing sends)
- Email reaches users regardless of app install status — useful for lapsed or churned user segments
