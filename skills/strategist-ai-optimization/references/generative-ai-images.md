## AI Image Generation (BrazeAI)

Generate images using AI directly within Braze's media library via DALL-E integration.

### Accessing the Generator

Navigate to **Templates > Media Library**, then select the **AI Image Generator** button (wand icon).

### How to Generate an Image

1. Open **Media Library** → click **AI Image Generator**
2. Enter a text description (up to 300 characters) — more detail produces better results
3. Click **Generate Images** — generation takes ~1 minute
4. Click the download icon on any image to save it to your media library

### Constraints

- Text-only prompts — uploading a reference image is not supported
- 300 character limit on descriptions
- Generated images are saved to the centralized media library for reuse across campaigns

### Media Library

All generated images are stored in **Templates > Media Library**, which serves as the single centralized asset repository across Braze.
