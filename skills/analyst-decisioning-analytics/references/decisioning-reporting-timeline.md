## Decisioning Timeline Reports

The timeline report provides a visual record of key events alongside performance metrics, with annotations appearing directly on uplift charts and in a dedicated timeline tab.

**Event types tracked:**
- Agent runs
- Configuration changes
- Guardrail updates
- System updates

### Timeline Event Fields

| Field | Description |
|-------|-------------|
| Date | When the event occurred |
| Type | Category: system update, agent run, or configuration change |
| Label | Name or identifier for the event |
| Details | Additional descriptive information |
| Visible in Charts | Whether the event appears in related charts |

### Usage

- Use the dropdowns to select comparison criteria for comparing performance between two groups.
- Events are annotated directly on uplift charts, providing immediate context for metric shifts without manual historical lookup.
- The dedicated **Timeline** tab lists all events in tabular form alongside the chart view.
