## Open Source Software Disclosure — Braze Services

**Effective:** February 5, 2025 (subject to change)

Braze Services incorporate third-party code under open-source licenses. The disclosure list is point-in-time and may be updated; it may not be complete.

### Key Legal Points

- Using Braze Services **without modifying** OSS components does **not** require you to grant IP rights or expose your source code.
- If you **modify** the embedded OSS, you become responsible for any resulting obligations. Braze disclaims all liability for such modifications.
