# Banner Analytics

Banner analytics provide campaign-level and message-level performance data for Banner campaigns in Braze.

## Accessing Banner Analytics

Navigate to your Banner campaign in the Braze dashboard and select the **Analytics** tab to review performance data.

## Key Analytics Areas

### Campaign Details
- Campaign name, channel, and status
- Send dates and scheduling configuration
- Audience targeting summary

### Message Performance
Core metrics tracked for Banner messages:
- **Impressions** — Number of times the banner was displayed to users
- **Clicks** — Number of user interactions with the banner
- **Click-through rate (CTR)** — Clicks divided by impressions
- **Dismissals** — Number of times users closed/dismissed the banner

### Historical Performance
- Time-series charts showing metric trends over the campaign lifetime
- Ability to filter by date range to compare performance across periods

## Notes

- Banner analytics follow the same general structure as other Braze campaign analytics
- Metrics are updated in near real-time in the dashboard
- For funnel analysis and conversion tracking, configure conversion events at the campaign level before launch
