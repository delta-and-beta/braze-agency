## Contributing to Braze Docs

Braze Docs accepts community contributions merged every **Tuesday and Thursday**.

**Requirements:**
- Basic Git knowledge required
- Contributions accepted in **English only**
- Must sign the [Contribution License Agreement (CLA)](https://www.braze.com/docs/cla) — unsigned PRs are auto-blocked by `@cla-bot`

## Setup Steps

1. Create a GitHub account at github.com/signup
2. Sign the CLA at braze.com/docs/cla
3. Fork the [braze-inc/braze-docs](https://github.com/braze-inc/braze-docs) repository (keep default fork settings)
4. Choose a contribution method:

| Method | Best For | Requirements |
|--------|----------|--------------|
| GitHub website | Small, single-document changes | None — ready to contribute immediately |
| Local environment | Complex or multi-document changes | Must set up local environment first |
