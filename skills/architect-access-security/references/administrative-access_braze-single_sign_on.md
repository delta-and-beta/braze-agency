## Single Sign-On Overview

Single Sign-On (SSO) support in Braze allows users to authenticate via centralized identity providers.

### Supported SSO Types

- **Google SSO** — OAuth-based authentication via Google accounts
- **SAML SSO** — Enterprise-grade federated authentication via SAML 2.0

### Availability

| SSO Type | Availability |
|----------|-------------|
| Google SSO | Pro and Enterprise plans |
| SAML SSO | Pro and Enterprise plans |
| Other plans | Contact Braze account manager |

### Key Points

- SSO is not available on all plans by default; non-Pro/Enterprise customers must contact their Braze account manager to enable it.
- Both Google and SAML SSO are gated at the same plan tier.
