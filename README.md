# braze-agency

Braze platform specialist agency for Claude Code. 9 AI agents backed by 166 skills and 1,304 topic references with semantic search.

## Quick Start

```bash
npm install @anthropic-apps/braze-agency
npx braze-agency install
```

Restart Claude Code. The agents appear automatically and respond to Braze-related questions.

### Semantic Search (MCP)

The `braze-agency` MCP server provides:
- `braze_search` — find relevant skills or topics by semantic similarity
- `braze_get_topic` — read a specific topic reference file
- `braze_list_skills` — list all 166 skills with descriptions

### Knowledge Base

- **166 skills** covering SDK setup, API integration, Canvas design, email infrastructure, analytics, and more
- **1,304 topic references** — atomic knowledge units synthesized from the Braze documentation
- **bge-m3 embeddings** (1024-dim) for high-quality semantic search

## Commands

```bash
npx braze-agency install    # Link agents + register MCP
npx braze-agency uninstall  # Remove agents + MCP
npx braze-agency status     # Check installation
```

## How It Works

1. `install` symlinks agent `.md` files to `~/.claude/agents/`
2. Registers an MCP server (`braze-agency`) for semantic search
3. Claude Code auto-discovers the agents and dispatches based on query content
4. Agents search the knowledge base via MCP to ground responses in Braze documentation

## Cross-Agency Collaboration

Install multiple agencies to enable cross-platform expertise:

```bash
npm install @anthropic-apps/braze-agency
npm install @anthropic-apps/marketo-agency  # hypothetical

npx braze-agency install
npx marketo-agency install
```

Ask Claude: *"Help me migrate from Marketo to Braze"* — it will dispatch both braze and marketo agents, each searching their own knowledge base.
